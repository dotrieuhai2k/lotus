# Chào mừng đến với Tài liệu Kỹ thuật Lotus

Đây là cổng thông tin chính thức về kiến trúc, vận hành và phát triển hệ thống **Lotus**.

## 🚀 Bắt đầu nhanh

* **Cho Lập trình viên:** Xem hướng dẫn [Thiết lập môi trường](development/setup.md).
* **Cho DevOps/SRE:** Xem các [Quy trình vận hành (Runbooks)](operations/runbooks.md).
* **Kiến trúc:** Tìm hiểu về [Luồng dữ liệu](architecture/overview.md).

## 🛠 Cấu trúc tài liệu

| Mục | Mô tả |
| :--- | :--- |
| **Architecture** | Sơ đồ hệ thống, mô hình dữ liệu. |

| **Operations** | Hướng dẫn xử lý sự cố, backup, restore. |
